# НейроУчитель
Это десктопное приложение на Python (CustomTkinter), которое помогает школьникам и студентам (может помочь даже преподавателям):

- **Генерировать тесты** по темам или по произвольному материалу;
- **Получать объяснения тем и понятий** в удобном чат-интерфейсе;
- **Получать помощь с домашними заданиями** в формате подсказок и пошаговых наводящих вопросов.

Приложение использует сторонние ИИ‑модели через библиотеку `g4f` и несколько провайдеров (Yqcloud, OpenRouter, DeepInfra, PollinationsAI).

Если вы использовали Win+Shift+S (скриншот экрана) или что-то подобное с Shift, то нужно ещё раз прожать Shift в программе.
Это делается чтобы списки в настройках/сообщения в чатах можно было листать.
После активации подобных комбинаций модуль интерфейса думает что Shift ещё зажат и не даёт листать элементы интерфейса.

## Возможности

- **Создание тестов**
  - Выбор сложности: лёгкий / средний / тяжёлый.
  - Настройка количества вопросов.
  - Выбор типов заданий: с выбором ответа, с вводом ответа.
  - Возможность запросить автоматическую генерацию объяснений к каждому заданию.

- **Чат «Объяснение темы / понятий»**
  - Режим дружелюбного преподавателя, который объясняет кратко и понятно.
  - Поддержка уточняющих вопросов и мини‑проверок понимания.

- **Чат «Помощь с ДЗ»**
  - Режим наставника по домашним заданиям.
  - Вместо прямого ответа даёт подсказки, направления и методы решения.

- **Работа с чатами**
  - Несколько чатов, переключение между ними.
  - Импорт/экспорт чатов в/из JSON‑файла.
  - Ограничение длины истории (по умолчанию до 25 сообщений) для стабильной работы.

- **Настройки приложения**
  - Выбор внешнего вида: тёмная, светлая, системная тема.
  - Выбор основной цветовой темы интерфейса.
  - Выбор ИИ‑модели.
  - Выбор провайдера ИИ-модели
  - Управление автосохранением/автозагрузкой чатов.
  - Включение/отключение иконки в системном трее.
  - Ввод API ключа для некоторых провайдеров

## Технологии

- **Python 3.9+** (рекомендуется актуальная версия Python 3);
- **CustomTkinter** — современный UI‑фреймворк для Tkinter;
- **PyStray** — иконка приложения в системном трее;
- **Pillow (PIL)** — работа с иконками и изображениями;
- **g4f** — работа с различными ИИ‑провайдерами и моделями;
- Дополнительные библиотеки: `CTkMessagebox`, `CTkCodeBoxPlus`, `CTkListbox`, `pyperclip`, `webbrowser`, стандартная библиотека Python.
